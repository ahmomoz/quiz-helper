# React Quiz Helper

A specialized interactive quiz application designed for React Senior Developer interviews. Built with modern web technologies and a "Neo-Brutalist" design aesthetic to make learning advanced React concepts engaging.

## Tech Stack

- **Framework:** [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Linting & Formatting:** ESLint + Prettier

## Features

- **Interactive Quiz:** Practice with advanced React interview questions.
- **Immediate Feedback:** Real-time validation of answers with visual cues.
- **Detailed Explanations:** Deep dive into concepts like Virtual DOM, Hooks, and Performance upon answering.
- **Neo-Brutalist Design:** A bold, high-contrast UI with hard shadows and vibrant colors.
- **Responsive:** Optimized for both desktop and mobile devices.

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- pnpm (v9 or higher recommended)

### Installation

1.  Clone the repository:

    ```bash
    git clone <repository-url>
    cd quiz-helper
    ```

2.  Install dependencies:
    ```bash
    pnpm install
    ```

### Development

Start the development server:

```bash
pnpm dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

### Building for Production

Build the application for deployment:

```bash
pnpm build
```

The output will be in the `dist` directory.

### Linting and Formatting

- Run linter: `pnpm lint`
- Format code: `pnpm format`
